# ✋ Multi-Finger Elastic Interaction

## 📌 Overview
This project uses OpenCV and MediaPipe to track both hands and draw stylized elastic lines between corresponding fingers.

## 🚀 Features
- Real-time hand tracking
- Multi-finger connections (5 fingers)
- Stylized glowing elastic lines
- Smooth motion tracking

## 🛠️ Tech Stack
- Python
- OpenCV
- MediaPipe

## ▶️ Run
pip install opencv-python mediapipe  
python main.py

## 🎯 Output
Interactive visual lines between fingers of both hands
